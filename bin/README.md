# Scripts de utilidad

Scripts auxiliares para el flujo de trabajo con paquetes.

## Scripts disponibles

- **`check-packages-sync.sh`** - Verifica que los paquetes locales estan sincronizados con GitHub
- **`composer-update-vcs.sh`** - Actualiza paquetes desde GitHub (modo VCS)
- **`reset-for-testing.sh`** - Reset de base de datos para testing de paquetes

## Instalacion

Para instalar Larafactu en produccion, usar el **wizard web**. Ver `docs/PRODUCTION_WEB_INSTALL.md`.

## Workflow

Ver `WORKFLOW.md` para el flujo de trabajo completo con paquetes.
